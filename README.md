# Wet Math Research Library 📚🌊

A structured research library for the Wet Math group — collaborative mathematics exploring the Groovy Commutator, Proportional Mathematics, Scale Geometry, and their deep connections to Grothendieck's programme.

## Structure

Every document follows a **tiered summary structure** (see [RULES.md](RULES.md)):

- **L0** (~100 tokens): One-sentence abstract — is this relevant?
- **L1** (~500-800 words): Structural overview — what's in here?
- **L2** (10-30% of original): Detailed summary — working reference
- **Full text**: Complete document in markdown
